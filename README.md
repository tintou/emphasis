# Emphasis
Render special effects in your videos with ease
## Dependencies
 * gobject-2.0
 * glib-2.0
 * gio-2.0
 * gee-0.8
 * gtk+-3.0
 * gstreamer-1.0
 * gstreamer-base-1.0
 * gstreamer-pbutils-1.0
 * gstreamer-video-1.0
 * gst-editing-services-1.0
## How to build
``` Shell
meson build --prefix=/usr
cd build
ninja
sudo ninja install
```
